# API Capitainerie de Russell

Application de gestion des réservations de catways pour le port de plaisance.

## Installation
1. `npm install`
2. Créer un fichier `.env` avec vos variables (MONGO_URL, SECRET_KEY)
3. `npm start`

## Documentation
- La documentation technique est disponible via l'interface : `/api-docs`
- Générer la documentation manuellement : `npm run doc`

## Fonctionnalités
- Gestion des utilisateurs (Auth JWT)
- Gestion des catways
- Gestion des réservations
